📋 SZYBKIE URUCHOMIENIE (30 sekund):
═══════════════════════════════════════════════════════════════

1. Otwórz terminal w folderze projektu
2. Uruchom: ./start.sh
   (LUB: docker-compose up --build)
3. Otwórz: http://localhost:3000
4. Zarejestruj się i testuj!

═══════════════════════════════════════════════════════════════

✨ FUNKCJONALNOŚCI:
═══════════════════════════════════════════════════════════════

✅ Logowanie i rejestracja (JWT + bcrypt)
✅ Dodawanie notatek (szyfrowanych AES-256)
✅ Edycja notatek
✅ Archiwizowanie notatek
✅ Usuwanie notatek
✅ Foldery z kolorami
✅ Tagi z kolorami
✅ Filtrowanie po folderach i tagach
✅ Wszystko szyfrowane end-to-end!
✅ Dark mode support
✅ Responsive design

═══════════════════════════════════════════════════════════════

🛠️ TECHNOLOGIE:
═══════════════════════════════════════════════════════════════

Frontend:  Next.js 14 + React + TypeScript
Styling:   Tailwind CSS
Backend:   Next.js API Routes
Database:  SQLite + Prisma ORM
Security:  AES-256 encryption + JWT + bcrypt
Deploy:    Docker + docker-compose

═══════════════════════════════════════════════════════════════

🔐 BEZPIECZEŃSTWO:
═══════════════════════════════════════════════════════════════

• Hasła hashowane (bcrypt, 10 rounds)
• Notatki szyfrowane AES-256-CBC
• Każda notatka ma unikalny IV
• JWT w httpOnly cookies
• Middleware zabezpieczający routes
